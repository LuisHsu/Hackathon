活動介紹
		
		
		
成大黑客松 — 飢餓遊戲

源自於賈伯斯於史丹佛大學的演講
"Stay hungry, stay foolish.”
以及黑客松的傳統文化—吃
飢餓代表的是一種渴望：
渴望有一個舞台能夠施展我們的創意

另外一層意義
是源自於黑客松雖然是一個創意創新競賽
但是卻是輕鬆的
大家可以在比賽過程中吃喝，結交朋友

黑客松 ( Hackathon ) 是由兩個字組合而成：
黑客 ( Hack ) ，馬拉松 ( Marathon )
這是一個讓參賽者在短期之內
專注於解決手邊問題
或是傾心於手邊專案而努力完成的競賽
因為在短期之內要做出產品雛型
所以需要仰賴高度的合作能力
以及創意思考能力

活動介紹
什麼是黑客松
 成大黑客松


黑客松的英文是 Hackathon 源自於 Hack + Marathon 。黑客松就是一種馬拉松式的程式創作活動。 參賽者必須在有限的時間內，利用手邊的資源來解決問題進而變成一個作品。在這樣的活動中，團隊合作和解決問題的能力是不可或缺的！



我們的黑客松將分成三個組別：
1.自由組
在自由組方面，我們會提供學校的開放資源
以及一些開放資料的網站讓大家 hack
不過沒有強制大家一定要使用。大家可以盡情發揮自己的創意！做出很酷的作品！

2.Maker 組
因為最近 Maker 風潮興起，提倡自己的東西自己做！
在這個組別考量到每一組別的需求不一樣
因此我們不會提供參賽者使用的硬體，需要參賽者自己攜帶。

3.企業組
考量到縮短產學差異
我們邀請到ＯＯＯ擔任我們的出題企業。
在這個組別中，你將使用企業提供的 API 或是平台來製作專案
並且會有企業 Mentor 一旁協助陪你成長！

此外，在活動的第一天我們將邀請到企業演講
Hack 累了不妨來聽一下演講，充電一下！
在活動的第二天雖然沒有演講可以聽
但是 Hack 累了可以來玩桌遊！
我們將開放一間桌遊室給大家盡情使用，桌遊之餘別忘記 Hack 囉！
